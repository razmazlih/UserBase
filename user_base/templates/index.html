רשימת הנתיבים של ה-API

תיאור נתיב מלא שיטה Header Body
מחזיר מידע על פרופיל המשתמש /api/users/profile/ GET Authorization: Bearer {jwt-token}
Content-Type: application/json 		לא נדרש
מעודכן את פרטי פרופיל המשתמש /api/users/profile/ PUT Authorization: Bearer {jwt-token}
Content-Type: application/json 
            {
                "phone_number": "0505050505",
                "city": "אשדוד",
                "street_name": "מנחם בגין",
                "house_number": 5
            }
יוצר משתמש חדש במערכת /api/users/register/ POST Content-Type: application/json 
                        {
                            "username": "example",
                            "password": "Password123",
                            "phone_number": "0505050505",
                            "city": "example",
                            "street_name": "example"
                          }
משנה את הסיסמה של המשתמש המחובר /api/users/change-password/ PUT Authorization: Bearer {jwt-token}
Content-Type: application/json 
    {
        "new_password": "Password123!"
    }
משנה את שם המשתמש של המשתמש המחובר /api/users/change-username/ PUT Authorization: Bearer {jwt-token}
Content-Type: application/json 
    {
        "new_username": "new_username"
    }
מקבל טוקן גישה (JWT) עבור התחברות /api/login/ POST Content-Type: application/json 
    {
        "username": "example",
        "password": "Password123!"
    }
רענון טוקן גישה בעזרת טוקן רענון /api/refresh/ POST Content-Type: application/json 
    {
        "refresh": "{refresh-token}"
    }
